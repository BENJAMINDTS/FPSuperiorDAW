# 🐘 Desarrollo Web en Entorno Servidor (DWES)

Bienvenido al repositorio de **PHP**. En este módulo profundizamos en el backend, aprendiendo a crear aplicaciones web dinámicas, seguras y escalables, gestionando la lógica de negocio y la persistencia de datos.

## 📚 Temario (Resultados de Aprendizaje)

El curso está estructurado en 9 unidades didácticas que se corresponden con los RAs oficiales:

### 🟢 Fundamentos del Servidor
- **[Tema 1 (RA01): Arquitecturas Web](./UD.01)**
  - Funcionamiento cliente-servidor y protocolo HTTP.
  - Configuración del entorno de desarrollo (XAMPP/LAMP).
  - Selección de herramientas y servidores de aplicaciones.

- **[Tema 2 (RA02): Sintaxis y Embebido](./UD.02)**
  - Integración de código PHP en HTML.
  - Tipos de variables, operadores y configuración del `php.ini`.
  - Ámbitos de utilización de variables.

- **[Tema 3 (RA03): Estructuras de Control](./UD.03)**
  - Bucles, condicionales y arrays (vectores/matrices).
  - **Formularios:** Procesamiento de datos enviados por el usuario.
  - Validación de entrada y creación de funciones reutilizables.

### 🟡 Desarrollo Avanzado y Seguridad
- **[Tema 4 (RA04): Gestión de Estado](./UD.04)**
  - **Sesiones y Cookies:** Mantenimiento de información del usuario.
  - Mecanismos de autenticación (Login/Logout).
  - Seguridad y control de acceso.

- **[Tema 5 (RA05): POO y Separación de Lógica](./UD.05)**
  - **Patrón MVC:** Separación de Modelo, Vista y Controlador.
  - Programación Orientada a Objetos en PHP (Clases, Objetos, Herencia).
  - Mantenimiento y documentación del código.

### 🔴 Datos, Servicios y Frameworks
- **[Tema 6 (RA06): Acceso a Datos](./UD.06)**
  - Conexión a Bases de Datos (PDO / MySQLi).
  - Operaciones CRUD (Crear, Leer, Actualizar, Borrar).
  - Seguridad contra inyección SQL.

- **[Tema 7 (RA07): Servicios Web](./UD.07)**
  - Creación y consumo de **APIs** (REST/SOAP).
  - Intercambio de información entre sistemas heterogéneos.
  - Estándares y protocolos web.

- **[Tema 8 (RA08): Aplicaciones Dinámicas](./UD.08)**
  - Generación de páginas dinámicas avanzadas.
  - Uso de frameworks del lado del servidor.
  - Modificación dinámica de contenido y estructura.

- **[Tema 9 (RA09): Aplicaciones Híbridas](./UD.09)**
  - Reutilización de código y repositorios.
  - Integración de librerías externas (Big Data, BI).
  - Despliegue de aplicaciones web híbridas.

 - **[Laravel: Primeros Proyectos Laravel](./Laravel)**
   -Creación de primeros proyectos de laravel
   -Consumo de apis y vistas basicas

- **Clean Code:** Nomenclatura clara y estructura modular.
