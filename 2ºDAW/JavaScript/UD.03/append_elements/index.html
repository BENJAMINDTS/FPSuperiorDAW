Ejercicio 4: Añadir elementos desde script

Cada elemento se genera al hacer clic en su botón correspondiente

1. Crear un párrafo

Crea un script que añada un párrafo al final del body con el texto: "Este párrafo fue creado con JavaScript".

2. Añadir un título dinámicamente

Escribe un script que cree un elemento <h2> con el texto "Bienvenido a mi página" y lo inserte dentro de un elemento con id "encabezado".

3. Insertar un botón en una sección

Crea un botón con el texto "Haz clic aquí" y añádelo dentro del elemento con id "seccion-botones".

4. Agregar varios párrafos

Crea un bucle que genere 3 párrafos con los textos "Párrafo 1", "Párrafo 2", "Párrafo 3" y los añada todos al body.

5. Lista de tareas

En un contenedor con id "lista-tareas", crea una lista <ul> con tres elementos <li> que contengan las tareas: "Estudiar", "Comer", "Dormir".

6. Generar elementos desde un array

Dado el siguiente array: ["Manzana", "Banana", "Naranja"], crea una lista <ul> y, usando un bucle, añade un <li> para cada fruta.

7. Insertar una imagen

Crea un elemento <img> con src="imagen.jpg" y alt="Foto de ejemplo", y añádelo al final del body.

8. Botón que crea un elemento nuevo

Crea un botón con el texto "Añadir párrafo". Al hacer clic, debe crear un nuevo <p> con el texto "Nuevo párrafo creado" y añadirlo debajo del botón.

9. Contador de clics

Crea un botón con el texto "Haz clic". Cada vez que el usuario lo pulse, se debe crear un nuevo párrafo que diga "Has hecho X clics", donde X es el número de veces que se ha pulsado.

10. Crear una tarjeta con varios elementos

Crea dinámicamente una "tarjeta" (<div>) que contenga: Un título (<h3>) con el texto "Tarjeta creada", un párrafo (<p>) con una descripción y una imagen (<img>).
