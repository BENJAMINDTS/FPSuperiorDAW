# ⚡ Desarrollo Web en Entorno Cliente (DWEC)

Bienvenido al repositorio de **JavaScript**. Este módulo se centra en dotar de interactividad y dinamismo a las interfaces web, gestionando el comportamiento del navegador y la comunicación asíncrona con el servidor.

## 📚 Temario (Resultados de Aprendizaje)

El curso cubre los 7 Resultados de Aprendizaje oficiales, estructurados en las siguientes unidades:

### 🟢 Fundamentos y Entorno
- **[Tema 1 (RA01): Arquitectura Cliente](./UD.01)**
  - Diferencias entre ejecución en cliente vs servidor.
  - Compatibilidad entre navegadores y estándares web.
  - Herramientas de depuración y desarrollo en el navegador.

- **[Tema 2 (RA02): Sintaxis Básica](./UD.02)**
  - Integración de scripts en HTML.
  - Tipos de datos, variables (let/const/var) y operadores.
  - Estructuras de control y bucles.

- **[Tema 3 (RA03): Objetos Predefinidos (BOM)](./UD.03)**
  - Uso de objetos nativos del lenguaje (Date, Math, String).
  - Interacción con el navegador (**BOM** - Browser Object Model): `window`, `navigator`, `location`.
  - Cuadros de diálogo y gestión de ventanas.

### 🟡 Programación Avanzada e Interacción
- **[Tema 4 (RA04): Estructuras y Funciones](./UD.04)**
  - **Funciones:** Definición, parámetros y funciones flecha.
  - **Arrays:** Métodos de manipulación y recorrido.
  - **POO:** Creación de objetos propios, métodos y patrones de diseño.

- **[Tema 5 (RA05): Eventos y Formularios](./UD.05)**
  - Manejo de **Eventos**: Click, teclado, carga, etc.
  - Validación de formularios en cliente (Expresiones Regulares).
  - Gestión del foco y comportamiento del usuario.

### 🔴 DOM y Comunicación Asíncrona
- **[Tema 6 (RA06): Manipulación del DOM](./UD.06)**
  - **DOM (Document Object Model):** Acceso y modificación del árbol HTML.
  - Creación y eliminación dinámica de nodos y elementos.
  - Separación de contenido, presentación y comportamiento.

- **[Tema 7 (RA07): Comunicación Asíncrona (AJAX)](./UD.07)**
  - Mecanismos de comunicación **AJAX** (XMLHttpRequest / Fetch API).
  - Consumo de APIs y gestión de respuestas JSON.
  - Actualización dinámica de contenido sin recargar la página.

### 🔵 JQuery

- **[JQuery](./JQuery)**
  - Selección de elementos y manipulación del **DOM** con jQuery.
  - Gestión de eventos, efectos y animaciones.
  - Comunicación asíncrona (**AJAX**) con jQuery.

### 🟢 Node.js

- **[Node.js](./Node)**
  - Módulos nativos de Node.js: `fs`, `http`.
  - Creación de módulos propios y exportación de funciones.
  - Servidor HTTP nativo con enrutamiento manual.
  - **Express.js:** rutas, parámetros de URL y peticiones POST con JSON.
  - API REST completa — CRUD de películas. Consulta el [README de Node](./Node/README.md) para instrucciones detalladas.
