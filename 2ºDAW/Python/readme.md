## 📚 Temario (Resultados de Aprendizaje)

El curso cubre 5 Resultados de Aprendizaje clave, desde la lógica básica hasta la persistencia de datos:

### 🟢 Introducción y Algoritmia
- **[Tema 1 (RA01): Conceptos Básicos](./UD.01)**
  - Componentes del lenguaje y paradigmas de programación.
  - Sintaxis básica y resolución de ejercicios teóricos.
  
- **[Tema 2 (RA02): Algoritmos](./UD.02)**
  - Diseño de algoritmos mediante **Diagramas de Flujo**.
  - Implementación de lógica y resolución de problemas simples.
  - Pruebas de funcionalidad y validación.

- **[Tema 3 (RA03): Estructuras de Control](./UD.03)**
  - Uso de condicionales y bucles.
  - Gestión de **Entrada/Salida** de datos.
  - Depuración de errores y documentación del código.

### 🟡 Estructuras de Datos
- **[Tema 4 (RA04): Colecciones](./UD.04)**
  - Manipulación avanzada de **Listas, Tuplas y Diccionarios**.
  - Uso de métodos integrados (`append()`, `pop()`, `keys()`, etc.).
  - Procesamiento de información con estructuras anidadas (listas de listas, diccionarios complejos).
  - Análisis de eficiencia en el manejo de datos.

### 🔴 Programación Avanzada
- **[Tema 5 (RA05): POO y Bases de Datos](./UD.05)**
  - **POO:** Clases, Objetos, Atributos y Métodos.
  - Implementación de Herencia y Encapsulación.
  - **Persistencia:** Conexión con Bases de Datos SQL (vía **SQLite3**).
  - Operaciones CRUD (Crear, Leer, Actualizar, Borrar) desde scripts.

### 🌐 Framework Web: Django

- **[Django: Aplicación Web con MVT](./Django)**
  - Aplicación práctica con **Django 6**, el framework web de alto nivel para Python.
  - Amplía los conocimientos de POO y bases de datos de RA05 en un entorno web completo.
  - Dos aplicaciones: `tienda` (ejercicios 3–14) y `peliculas` (proyecto final CRUD).
  - Patrón **MVT**, herencia de plantillas, panel de administración y diseño Mobile-First.

## 🧩 Buenas Prácticas
En este repositorio se aplican convenciones propias de Python:
- **PEP 8:** Estilo de código estándar (indentación, nombrado de variables).
- **Docstrings:** Documentación interna de funciones y clases.
- **Tipado Dinámico:** Gestión correcta de tipos de datos.
