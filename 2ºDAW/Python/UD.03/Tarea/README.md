# RA03 - Ejercicios Python

## Requisitos del sistema

- Python 3.8 o superior

## Estructura del Proyecto

Tarea/
├── RA03_A/
│   ├── ejercicio1.py
│   └── ejercicio2.py
├── RA03_B/
│   └── ejercicio3.py
├── RA03_C/

└── RA03_D/

## Descripción de Ejercicios

### RA03_A

- Ejercicio1.py: Encuentra el número mayor entre tres números ingresados por el usuario
- Ejercicio2.py: Calculadora básica con operaciones +, -, *, /

### RA03_B

- Ejercicio3.py: Muestra todos los números desde 1 hasta un número N ingresado

### RA03_C

- Ejercicio4.py: Calcula el área de un rectángulo solicitando base y altura

### RA03_D

- Ejercicio5.py: Compara el rendimiento entre bucle for y función sum() para sumar números del 1 al 1,000,000

## Uso

Ejecutar cada ejercicio individualmente:

python RA03_A/ejercicio1.py
python RA03_D/ejercicio5.py
